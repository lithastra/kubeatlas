# KubeAtlas Governance

This document is a placeholder. The full governance model — covering
decision-making, maintainer nomination, working groups, and conflict
resolution — will be ratified before applying to CNCF Sandbox (planned
for milestone M6, after v1.0 release).

## Current state (Pre-Alpha, Phase 0)

The project currently has a single primary maintainer (see
[MAINTAINERS.md](./MAINTAINERS.md)) who has final say on all technical
and project-direction decisions. Pull requests require maintainer
approval to merge.

## Principles

Even in this early stage, the project follows these principles:

1. **Open by default**: All technical discussions happen on GitHub
   issues / PRs / Discussions. No private decision-making about the
   public roadmap.
2. **DCO for all contributions**: Every commit must carry a
   Signed-off-by line.
3. **Conventional Commits**: All commit messages and PR titles follow
   the [Conventional Commits](https://www.conventionalcommits.org/)
   specification.
4. **English for shared artifacts**: Code, comments, commit messages,
   PR/Issue titles are English. Documentation is English-canonical;
   translations under `docs/<locale>/` are welcome.
